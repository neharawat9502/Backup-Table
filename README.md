# Backup-Table
Task 1: Excel Basics - Data Cleaning & Formatting
Project Overview
This project involves cleaning and standardizing a messy dataset to prepare it for analysis.

Key Steps Taken

Data Cleaning: Identified missing values and handled duplicates to prevent data loss or inaccuracy.


Standardization: Used functions like TRIM  to fix inconsistent naming and remove extra spaces.



Formatting: Validated date and numeric columns to ensure correct data types.



Data Quality: Added a Data_Quality_Notes column to document specific cleaning decisions.
